# Odoo Sales CRM Module

A custom CRM module for Odoo designed to streamline customer relationship management and sales operations. This module enables businesses to manage leads, track opportunities, monitor sales pipelines, schedule follow-up activities, and gain valuable insights through interactive dashboards and reports.

## Features

* Lead and Opportunity Management
* Customer & Contact Management
* Sales Pipeline with Kanban View
* Activity Scheduling & Follow-ups
* Sales Dashboard & KPIs
* Customer Interaction History
* Security Groups & User Permissions
* Search, Filters & Group By
* Automated Business Workflows
* Reporting and Analytics

## Tech Stack

* Python
* Odoo Framework
* Odoo ORM
* XML
* PostgreSQL
* Docker

## Project Goals

This project demonstrates how to build modular ERP applications using Odoo's framework while following clean architecture and best practices. It showcases custom models, business logic, XML views, security rules, workflows, and reporting to solve real-world CRM challenges.
